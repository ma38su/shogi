# Shogi38

将棋盤プロトタイプ - AI対戦機能付き

## デモ

[GitHub Pages](https://ma38su.github.io/shogi/)

## セットアップ

```sh
npm install
```

## 開発

```sh
npm run dev
```

ブラウザで表示される URL を開いてください。

## ビルド

```sh
npm run build
```

`dist/` ディレクトリに静的ファイルが出力されます。

## 技術スタック

- [Vite](https://vite.dev/) - ビルドツール
- [React](https://react.dev/) - UI ライブラリ
- [Tailwind CSS](https://tailwindcss.com/) - スタイリング
- TypeScript
